Embedded Linux
===============

This week, the embedded Linux landscape has seen a mix of hardware developments, software updates, and community discussions, showcasing innovation and ongoing challenges in the field.

Hardware Developments
---------------------

Notable advancements in hardware, particularly in single-board computers, highlight the ongoing evolution in embedded systems.

- **Dragon Q8B SBC combines Snapdragon 8cx Gen 3 with dual 2.5GbE**: Radxa has introduced the Dragon Q8B, a compact single-board computer built around the Qualcomm Snapdragon 8cx Gen 3. Read more at `LinuxGizmos.com: embedded Linux news & devices <https://linuxgizmos.com>`_.

Software Updates and Improvements
----------------------------------

The embedded Linux community continues to refine and expand its software offerings, with important updates and enhancements discussed this week.

- **meta-kiss: updates to our simple Yocto layer**: Bootlin shares insights on simplifying Yocto through updates to the meta-kiss project, making embedded development more accessible. Find details at `News – Bootlin <https://bootlin.com/blog/category/news>`_.
- **The 7.1 kernel has been released**: Linus Torvalds has officially released the 7.1 kernel, marking another significant milestone for the Linux community. Learn more at `The 7.1 kernel has been released <https://lwn.net/Articles/1077758/>`_.
- **Revised AVX-512 xor_gen() Implementation For Linux RAID Yielding More Performance Gains**: A new implementation for the Linux kernel function aimed at improving performance in RAID configurations has been reported by Phoronix. Check out the article at `Revised AVX-512 xor_gen() Implementation For Linux RAID Yielding More Performance Gains <https://www.phoronix.com/news/AVX-512-Xor-Gen-More-Perf>`_.
- **pkgcli As PackageKit's Modern, Nicer Command Line Interface**: Matthias Klumpp introduces pkgcli, a modern command-line interface for PackageKit, enhancing package management in Linux. Read the full post at `pkgcli As PackageKit's Modern, Nicer Command Line Interface <https://www.phoronix.com/news/PackageKit-pkgcli>`_.
