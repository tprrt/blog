About

About me

I am Thomas Perrot, an embedded software engineer based in Toulouse, France. My work centres on the Linux kernel, the Yocto Project / OpenEmbedded ecosystem, Zephyr RTOS, and open-source hardware. I contribute patches to upstream projects and occasionally speak at events such as Capitole du Libre.

You can find a full list of my open-source contributions on the Contributions page, download my Curriculum vitae, or visit my LinkedIn profile.

About this blog

This blog is a place where I write about topics I find interesting:

  • Embedded Linux — Yocto Project releases, toolchain notes, board bring-up.
  • Zephyr RTOS — device drivers, Bluetooth, build system.
  • Open-source hardware — single-board computers, FPGA, retro-computing.
  • Linux gaming — native ports, Proton, SteamOS, emulation.
  • Retro gaming — hardware preservation, game engines, homebrew.
  • Competitive fitness — CrossFit, Hyrox, powerlifting.

Long-form articles appear under Articles. The weekly Digests are short link roundups covering the same topics.
